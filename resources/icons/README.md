# Icons Directory

Thư mục này chứa các icon và logo cho ứng dụng.

## Logo

- `logo.jpg`: Logo chính của ứng dụng (Quân khu 1)

Logo được sử dụng cho:
- Window icon (icon hiển thị trên thanh taskbar và title bar)
- Splash screen khi khởi động ứng dụng
- Application icon

## Thêm icon mới

Để thêm icon mới, đặt file vào thư mục này và cập nhật code trong:
- `src/main.py`: Để set application icon và splash screen
- `src/ui/main_window.py`: Để set window icon
